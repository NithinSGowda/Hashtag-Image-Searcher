API Documentation

Use our hashtag search API to get urls of images across popular social media sites. Our API will return you a pure JSON response of direct image urls.

HOW TO USE OUR API ?

Query string = "http://i-search.ml/api?q={Your_Query_String_goes_here)}"

Access method "GET"

Parameters : q = The query tag on the images you need
Example code to get images related to flowers : "http://i-search.ml/api?q=flowers"

How to Implement on your code :

1] Python :


import http.client
conn = http.client.HTTPConnection("i-search.ml")
headers = {
'cache-control': "no-cache",
'postman-token': "043ba463-49eb-6678-0300-32e0cbe1a1c1"
}
conn.request("GET", "/api?q=flowers", headers=headers)
res = conn.getresponse()
data = res.read()
print(data.decode("utf-8"))

3] NodeJS :


var http = require("http");
var options = {

"method": "GET",
"hostname": "i-search.ml",
"port": null,
"path": "/api?q=flowers",
"headers": {
"cache-control": "no-cache",
"postman-token": "4e87e97a-7965-46d9-8acc-3efe0d54f996"
}
};
var req = http.request(options, function (res) {
var chunks = [];
res.on("data", function (chunk) {
chunks.push(chunk);
});
res.on("end", function () {
var body = Buffer.concat(chunks);
console.log(body.toString());
});
});
req.end();
